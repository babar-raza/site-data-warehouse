GSC Warehouse - Available Scripts
==================================

Root Level Scripts:
-------------------

Deployment:
  deploy.sh / deploy.bat              - Initial deployment
  redeploy.sh / redeploy.bat          - Update/redeploy system

Operations:
  collect-stats.sh / .bat             - Display warehouse statistics
  manual-collection.sh / .bat         - On-demand data collection
  backup.sh / backup.bat              - Create database backup
  restore.sh / restore.bat            - Restore from backup

deployment/scripts/:
--------------------

  health-check.sh / .bat              - Check service health
  start-collection.sh / .bat          - Start data collection
  stop.sh / .bat                      - Stop all services
  validate-setup.sh / .bat            - Validate deployment setup

Usage Examples:
---------------

Windows:
  deploy.bat
  redeploy.bat --skip-data-collection
  collect-stats.bat
  manual-collection.bat --days 7
  backup.bat
  restore.bat backups\gsc_warehouse_20241113.sql.gz

Linux/Mac:
  ./deploy.sh
  ./redeploy.sh --skip-data-collection
  ./collect-stats.sh
  ./manual-collection.sh --days 7
  ./backup.sh
  ./restore.sh backups/gsc_warehouse_20241113.sql.gz
